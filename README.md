# Concerto

Real-time collaborative music education platform with video conferencing and audio editing.

## Features

- **Video rooms** — LiveKit-powered video/audio conferencing with room keys and waiting list approval
- **Audio editor** — Multi-track recording, trimming, volume/playback control, waveform visualization
- **Role-based access** — Admin, moderator, student, and participant roles with promotion/demotion
- **File management** — Upload, organize, and share audio files in folders
- **Moderation** — Participant approval, kick, and room mode controls

## Tech Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS v4 · LiveKit · PostgreSQL · Drizzle ORM · Zustand · XState · Better Auth · Bun
