# Concerto

Real-time collaborative music education platform with video conferencing and role-based classroom moderation.

## Features

- **Video rooms** — RealtimeKit-powered video/audio conferencing with room keys and waiting list approval
- **Role-based access** — Admin, moderator, student, and participant roles with promotion/demotion
- **File management** — Upload, organize, and share files in folders
- **Moderation** — Participant approval, kick, and room mode controls

## Tech Stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS v4 · RealtimeKit · PostgreSQL · Drizzle ORM · Zustand · Better Auth · Bun
